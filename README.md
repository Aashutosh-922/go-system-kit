# Go System Kit

Reusable Go utilities for building distributed systems.

Includes:

- Worker Pool
- Retry with Backoff
- Graceful Shutdown
- Event Bus
- Rate Limiter
- Circuit Breaker
- Trace Context
- Generic LRU Cache
- Concurrency-Safe Message Queue
- Mini Gin-Style Middleware Framework

Examples:

- `go run ./examples/lrucache`
- `go run ./examples/messagequeue`
- `go run ./examples/minigin`
